# نظام السلف النقدية

مشروع Next.js + Prisma + PostgreSQL لإدارة طلبات السلف وتسويتها.

## التشغيل المحلي

1. انسخ ملف البيئة:
   - انسخ `.env.example` إلى `.env`
2. عدل `DATABASE_URL` حسب قاعدة البيانات.
3. ثبت الحزم:
   - `npm install`
4. نفذ أوامر Prisma:
   - `npx prisma generate`
   - `npx prisma db push`
5. شغل المشروع:
   - `npm run dev`

## النشر على Vercel

1. ارفع المشروع إلى GitHub.
2. أنشئ مشروعاً جديداً في Vercel.
3. اربط المستودع.
4. أضف متغير البيئة `DATABASE_URL` في إعدادات Vercel.
5. استخدم قاعدة PostgreSQL خارجية.
6. نفذ بعد أول نشر:
   - `npx prisma db push`

## ملاحظات

- ملف `.env` غير مرفوع داخل المشروع.
- تم إضافة `.env.example` بدلاً منه.
- تم تضمين بنود السلفة داخل نموذج الطلب حتى يعمل نموذج التسوية بشكل صحيح.
