# نظام السلف النقدية

مشروع Next.js + Prisma + PostgreSQL لإدارة طلبات السلف وتسويتها.

## التشغيل المحلي

1. انسخ ملف البيئة:
   - انسخ `.env.example` إلى `.env`
2. عدل `DATABASE_URL` و`DIRECT_URL` حسب قاعدة البيانات.
3. ثبت الحزم:
   - `npm install`
4. نفذ أوامر Prisma:
   - `npx prisma db push`
5. شغل المشروع:
   - `npm run dev`

## النشر على Vercel

1. ارفع المشروع إلى GitHub.
2. أنشئ مشروعاً جديداً في Vercel.
3. اربط المستودع.
4. أضف متغيري البيئة `DATABASE_URL` و`DIRECT_URL` في إعدادات Vercel.
5. استخدم قاعدة PostgreSQL خارجية.
6. بعد أول نشر، نفذ مزامنة المخطط مع قاعدة البيانات من بيئة آمنة:
   - `npx prisma db push`
7. لا تضع `prisma db push` داخل أمر `build` في Vercel حتى لا يتم تعديل قاعدة البيانات تلقائياً أثناء كل عملية نشر.

## ملاحظات

- ملف `.env` غير مرفوع داخل المشروع.
- تم إضافة `.env.example` بدلاً منه.
- يجب تدوير أي بيانات اتصال قديمة إذا سبق رفعها داخل المستودع أو مشاركتها.
